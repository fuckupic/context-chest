## Recording Tips

- **Don't rehearse too much** — stumbling slightly is fine, it feels real
- **Use your actual voice**, not a "YouTuber voice"
- **Screen record with face cam** in the corner if possible — builds trust
- **Edit only for length** — cut pauses, don't add transitions or effects
- **Background music:** optional, very quiet if any
- **Total time:** aim for 60-75s after cutting dead air

This version works better for Reddit and Twitter. The polished version (video-script.md) works for the landing page. Record both if you can — same session, two takes.
